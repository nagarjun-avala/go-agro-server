# Go-Agro Server

## Server
- development
    ```
    npm run dev
    ```
- production
    ```
    npm start
    ```

## API Routes
- `/` for **Documentation**
- `/api/auth` for **Authentication Routes**
    ### Authentication Routes
    - `/api/auth/register` for **Register User**
    - `/api/auth/login` for **Login User**
    - `/api/auth/logout` for **Logout User**
    - `/api/auth/refresh_token` for **Refresh_token User**
